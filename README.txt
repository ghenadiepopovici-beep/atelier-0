
ATELIER — статическая PWA (демо, оффлайн).

Как развернуть (GitHub Pages):
1) Загрузите содержимое папки в новый репозиторий GitHub (файлы должны лежать в корне).
2) В Settings → Pages включите публикацию для ветки main / root.
3) Через минуту получите ссылку https://<user>.github.io/<repo>/

Как развернуть (Vercel):
1) Подключите GitHub к Vercel, затем Import Project → выберите репозиторий → Deploy.

Установка как приложение:
Откройте сайт в Chrome на телефоне → Меню → "Добавить на главный экран".
